# poeditor-sync

Download files from [poeditor.com](https://poeditor.com).

## Installation

```
go get github.com/frozzare/poeditor-sync
```

## Usage

Edit config file with api token, project id and files.

```
poeditor-sync --config=/path/to/config.json
```
